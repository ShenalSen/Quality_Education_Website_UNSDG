Task 1 - Gallery

Design Implementation: I meticulously crafted the gallery page to meet the
specifications. I ensured that each thumbnail was interactive, visually
indicating to users that they are clickable. Upon clicking, I programmed
the thumbnails to expand seamlessly into extended views,
revealing more detailed content associated with each image.


User Experience Enhancement: To enhance the user experience, I paid close attention to the details. I incorporated hover effects on
the thumbnails to make them visually inviting, prompting users to interact with them. Additionally, I ensured smooth transitions between
the thumbnail and extended view, creating a seamless browsing experience for our users.


Accessibility Considerations: Accessibility is crucial, so I made sure that all images and descriptions included appropriate alternative text.
This ensures that users with disabilities, such as those using screen readers, can still access and understand the content of our gallery page.


Personalization Options: Recognizing the importance of personalization, I provided users with customization options for the gallery page.
Users can now personalize their viewing experience by choosing their preferred color scheme and font within the extended view, adding a
touch of individuality to their interaction with our website.

Task 2 - Site Map

Design Implementation: In the development of the sitemap, I meticulously crafted a visually appealing representation of the website's
structure using Scalable Vector Graphics (SVG). Each page was accurately depicted as a node within the SVG canvas, with parent
and child pages represented by main branches and sub-branches, respectively.


User Experience Enhancement: Attention to detail was key in enhancing the user experience of the sitemap. I ensured that each SVG node
changed color on hover, making it visually interactive and inviting for users to explore. Additionally, I made sure that each node acted as a
link to the corresponding page it represented, allowing for seamless navigation.
